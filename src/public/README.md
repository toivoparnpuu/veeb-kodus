# Õppekava JSON andmete kuvamine

## Elemendid mida soovin näha

* Subject
* StudyYearNumber
* autum / spring
* optional / mandatory

### Filtrid

* optional
* studyYearNumber
* autumn
* spring
* assessment
* credit

## API pöörduspunktid

### /api/v1/curriculum

Sisaldab kõiki õppekava andmeid

### /api/v1/modules

Sisaldab kõiki mooduleid

### /api/v1/modules/:id

Sisaldab ühte moodulit

### /api/v1/subjects

Sisaldab kõiki aineid

### /api/v1/subjects/:id

Sisaldab ühte ainet
